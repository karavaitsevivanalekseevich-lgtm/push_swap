# Push_swap - Python

## Описание

Программа push_swap сортирует стек целых чисел, используя только допустимые операции со стеками (sa, sb, pa, pb, ra, rb, rra, rrb и их комбинации). Цель — отсортировать стек A в порядке возрастания за минимальное количество операций.

## Алгоритм

В программе используются разные подходы в зависимости от количества чисел:

**3 числа:** Перебор всех 6 возможных комбинаций — для каждой заранее определена оптимальная последовательность операций.

**4–5 чисел:** Алгоритм основан на сортировке трёх. Сначала два наименьших элемента пушатся в стек B, затем оставшиеся три сортируются алгоритмом для трёх, после чего минимумы возвращаются на вершину стека A.

**Больше 5 чисел (Butterfly):** Все элементы из стека A перекидываются в стек B, формируя структуру «бабочки». Алгоритм разделяет числа относительно счётчика: те что меньше — пушит вниз стека B, те что больше — оставляет наверху. Благодаря такой структуре максимальный элемент всегда находится близко к краям стека B. На втором этапе максимумы последовательно извлекаются из B и пушатся в A, формируя отсортированный стек.

## Результаты

- 3 числа: максимум 2 операции
- 5 чисел: максимум 12 операций
- 100 чисел: ~580 операций (лимит 700)
- 500 чисел: ~5080 операций (лимит 5500)

## Установка и использование

```bash
# Запуск через интерпретатор Python
python3 pushswap.py 3 1 2

# Или сделать файл исполняемым
chmod +x pushswap.py
./pushswap.py 3 1 2

# Подсчёт операций
python3 pushswap.py 3 1 2 | wc -l
```

## Структура проекта

- `pushswap.py` — точка входа, обработка аргументов и выбор алгоритма
- `stack.py` — реализация стека и операций (sa, ra, pb и т.д.)
- `validator.py` — валидация входных данных
- `small_sort.py` — сортировка для 3–5 элементов
- `big_sort.py` — алгоритм бабочки для больших наборов
